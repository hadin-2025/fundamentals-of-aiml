# fundamentals-of-aiml

Customer Segmentation Using Basic Machine Learning (Without Libraries)

This project focuses on segmenting customers based on their income and spending behavior using simple machine learning logic implemented in pure Python without any external libraries. The system calculates average income and spending values from the dataset and uses these as dynamic thresholds to classify customers into categories such as High Value, Wealthy, Big Spender, and Low Value. This approach demonstrates how basic data-driven decision-making can be achieved without complex algorithms, making it easy to understand the core concepts of customer segmentation and machine learning.
